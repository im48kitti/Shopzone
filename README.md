🛒 ShopZone — Full-stack E-commerce Platform
A robust online shopping platform featuring member registration, seat-map booking, and an advanced admin dashboard.

📋 Overview
ShopZone is a full-stack web application designed to provide a seamless user experience for online shopping and ticket booking. This project demonstrates my proficiency in Frontend Development, Database Management, and Software Quality Assurance.

✨ Key Features
User Management: Secure registration, login system, and profile management.

Dynamic Booking System: Tiered pricing and seat-map batch booking logic.

Payment Integration: Slip upload for payment verification with an automated QR Code generation system.

Admin Dashboard: Comprehensive control panel for user management, showtime scheduling, and sales reporting.

Quality Focused: Implemented Unit Testing and Manual Test Cases to ensure system reliability.

🛠 Tech Stack
Frontend: HTML5, CSS3, JavaScript (ES6+), React.js (Optional - based on your resume)

Backend: PHP / Node.js

Database: MySQL / MongoDB

Design: Figma (UI/UX)

Version Control: Git / GitHub

🧪 Testing & Quality Assurance
To ensure the application meets professional standards, I performed:

Unit Testing: Verified core functions like price calculation and seat availability logic.

Test Case Design: Created detailed test cases for Critical Path (Checkout, Payment, Registration).

UI/UX Testing: Ensured Responsive Design across various devices (Mobile, Tablet, Desktop).
